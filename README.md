# Student Performance Analysis 📊

## About
Analysed a student dataset of 20 students to find performance patterns.

## Tools Used
- Python
- NumPy
- Pandas

## Key Findings
- Top 5 performers were all female students
- Students studying 5+ hours scored 30 points higher on average
- 6 students identified as at-risk needing intervention
- 70% overall pass rate

## Skills Demonstrated
- Data cleaning and transformation
- Boolean indexing and filtering
- GroupBy aggregation
- Conditional column creation with .loc
